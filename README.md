// Notification System Extension - Push Notifications

// Assignment Description
This project extends an existing notification system by implementing a new `PushNotification` class that adheres to the `NotificationService` interface. This demonstrates the use of polymorphism and interface implementation in Java.

// Files Included
1.  NotificationService.java: The core interface.
2.  PushNotification.java: The new class implementing the push notification functionality.
3.  EmailNotification.java: Existing implementation for testing.
4.  SMSNotification.java: Existing implementation for testing.
5.  Main.java: Driver class to test all notification types.

// How to Run
1.  Ensure you have Java installed.
2.  Compile the files: "javac" ,"java"
3.  Run the main class: "java Main"
